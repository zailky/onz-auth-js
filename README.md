# onz-auth-js
Onz Auth JavaScript SDK

# Install

```shell
yarn add onz-auth or npm install onz-auth --save
```

or

```html
<script src="https://unpkg.com/onz-auth@1.0.2/dist/onz-auth-js-sdk.js"></script>
```
